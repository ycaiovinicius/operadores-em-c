# 🧠 Operadores em C

> Exercício prático desenvolvido para reforçar conceitos fundamentais da linguagem C.

---

## 🚀 Sobre o projeto

Este projeto demonstra o uso dos principais operadores da linguagem C:

* ➕ Operadores aritméticos
* 📝 Operadores de atribuição
* 🔁 Incremento e decremento

O objetivo é consolidar a lógica básica e entender como os dados são manipulados durante a execução do programa.

---

## 💻 Tecnologias utilizadas

* Linguagem C
* Compilador padrão

---

## 📌 Conceitos aplicados

✔ Soma, subtração, multiplicação e divisão
✔ Operadores `+=` e `*=`
✔ Operadores `++` e `--`
✔ Manipulação de variáveis
✔ Saída de dados com `printf`

---

## 📂 Estrutura do código

O programa:

1. Define valores iniciais
2. Realiza operações matemáticas
3. Aplica operadores de atribuição
4. Executa incremento e decremento
5. Exibe os resultados no terminal

---

## 🖥️ Exemplo de saída

```bash
Soma: 55
Subtração: 45
Multiplicação: 250
Divisão: 10
Novo Valor de X (Após += 10 e ++) 61
Novo Valor de Y (Após *= 5 e --) 24
```

---

## 🎯 Objetivo

Este exercício foi desenvolvido com foco em:

* Praticar lógica de programação
* Entender operadores em C
* Desenvolver base sólida para projetos mais avançados

---

Este é um projeto simples, mas essencial para construir uma base em programação.
